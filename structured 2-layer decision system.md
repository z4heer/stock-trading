# OVERVIEW: Two-Layer Stock Review System

```
Layer 1: Fundamental Review → Decide HOLD or EXIT candidate
Layer 2: Technical Review → Decide WHEN to exit or hold
```

Fundamentals tell **WHAT to hold**
Technicals tell **WHEN to hold or exit**

---

# PART 1: FUNDAMENTAL REVIEW (Monthly or Quarterly)

Tool: Screener.in, Zerodha Console, Investing.com

This step filters **good vs bad companies**

---

## STEP 1: Revenue Growth Check

Tool: Screener.in → Open stock → Profit & Loss

Criteria:

| Revenue trend           | Decision       |
| ----------------------- | -------------- |
| Increasing consistently | GOOD → Hold    |
| Flat for many quarters  | WARNING        |
| Decreasing              | EXIT candidate |

Example GOOD:

```
Year    Sales
2022    1000 cr
2023    1200 cr
2024    1500 cr
```

Means company growing.

---

## STEP 2: Profit Growth Check (MOST IMPORTANT)

Check Net Profit trend

Criteria:

| Profit trend | Decision       |
| ------------ | -------------- |
| Increasing   | Hold           |
| Flat         | Neutral        |
| Decreasing   | Exit candidate |

Profit matters more than revenue.

---

## STEP 3: EPS Growth Check

EPS = Earnings per share

Tool: Screener → EPS section

Criteria:

| EPS trend  | Decision       |
| ---------- | -------------- |
| Increasing | Strong hold    |
| Flat       | Neutral        |
| Decreasing | Exit candidate |

Institutional investors follow EPS heavily.

---

## STEP 4: Debt Level Check

Tool: Screener → Balance Sheet

Check:

Debt to Equity ratio

Criteria:

| Debt/Equity     | Decision       |
| --------------- | -------------- |
| Less than 0.5   | Excellent      |
| 0.5–1           | Acceptable     |
| Above 1         | Risky          |
| Increasing debt | Exit candidate |

---

## STEP 5: Return Ratios Check

Tool: Screener → Ratios

Check:

• ROCE
• ROE

Criteria:

| Ratio | Good      |
| ----- | --------- |
| ROCE  | Above 15% |
| ROE   | Above 15% |

Below 10% = weak company

---

## STEP 6: Promoter Holding Check

Tool: Screener → Shareholding

Criteria:

| Condition                   | Meaning  |
| --------------------------- | -------- |
| Promoter holding increasing | Positive |
| Stable                      | Neutral  |
| Decreasing                  | Negative |

Promoter selling = warning sign

---

## STEP 7: FII and DII Activity

Tool: Screener or Investing.com

Criteria:

| Condition      | Meaning |
| -------------- | ------- |
| FII increasing | Bullish |
| FII decreasing | Bearish |

Institutions move stock.

---

## STEP 8: Quarterly Result Check (VERY IMPORTANT)

Check last 2 quarters:

Compare:

• Sales growth YoY
• Profit growth YoY

If both falling → exit candidate

---

# FUNDAMENTAL DECISION SUMMARY

Continue HOLD if:

• Sales increasing
• Profit increasing
• EPS increasing
• Low debt
• ROCE > 15%
• Promoter holding stable/increasing

EXIT candidate if 3 or more negative.

# STEP 9: Trend Check

Add indicators:

• 20 EMA
• 50 EMA

Decision:

| Condition            | Action  |
| -------------------- | ------- |
| Price above both EMA | Hold    |
| Price below 20 EMA   | Warning |
| Price below 50 EMA   | Exit    |

---

# STEP 10: Support Level Check

Mark recent swing low.

If price breaks support → exit.

---

# STEP 11: RSI Check

Add RSI (14)

Decision:

| RSI      | Action         |
| -------- | -------------- |
| Above 60 | Hold           |
| 50–60    | Neutral        |
| Below 50 | Exit candidate |

---

# STEP 12: Volume Check

Exit if:

• High volume red candle
• Distribution pattern

Hold if:

• High volume green candles

---

# STEP 13: Institutional Activity Check (Advanced)

Tool: Sensibull

Check Open Interest

Exit if:

• Short buildup
• Call OI heavy above price

Hold if:

• Long buildup
• Put OI support

---

# FINAL DECISION MATRIX (PROFESSIONAL METHOD)

| Fundamentals | Technical | Decision     |
| ------------ | --------- | ------------ |
| Strong       | Strong    | HOLD         |
| Strong       | Weak      | Partial exit |
| Weak         | Weak      | EXIT         |
| Weak         | Strong    | Exit on rise |
