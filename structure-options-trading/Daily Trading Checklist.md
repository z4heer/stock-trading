# NIFTY Options System — Daily Trading Checklist

This checklist ensures every trade follows the **same structured process**.

Goal:

```text
Observation → Classification → Decision → Execution
```

Never skip steps.

---

# 1. Chart Setup

Use a NIFTY chart on TradingView or broker platform.

Timeframes:

| Timeframe | Purpose          |
| --------- | ---------------- |
| 15m       | Market structure |
| 5m        | Entry timing     |

Indicators:

| Indicator | Purpose               |
| --------- | --------------------- |
| VWAP      | Institutional bias    |
| EMA 20    | Momentum              |
| EMA 50    | Trend                 |
| Volume    | Breakout confirmation |

---

# 2. Mark Key Levels

Before trading mark these levels.

| Level             | Purpose            |
| ----------------- | ------------------ |
| Previous Day High | Breakout liquidity |
| Previous Day Low  | Stop liquidity     |
| Recent Swing High | Resistance         |
| Recent Swing Low  | Support            |

These levels often trigger **liquidity sweeps**.

---

# 3. Market Phase Identification

Check the market around **9:45 AM**.

Classify phase.

| Phase       | Description             |
| ----------- | ----------------------- |
| Expansion   | Strong directional move |
| Pullback    | Retracement toward EMA  |
| Compression | Tight range             |
| Transition  | Reversal signals        |

Write down:

```text
Market Phase = ______
```

---

# 4. Liquidity Sweep Check

Ask:

```text
Did price sweep a key level?
```

Examples:

Support sweep:

```text
Support = 22400
Price dips to 22390
Price reverses upward
```

Resistance sweep:

```text
Resistance = 22500
Price spikes to 22520
Price drops downward
```

Record:

```text
Setup = Support Sweep / Resistance Sweep
```

---

# 5. Determine Market Bias

Look at price relative to VWAP.

| Condition             | Bias    |
| --------------------- | ------- |
| Price above VWAP      | Bullish |
| Price below VWAP      | Bearish |
| Frequent VWAP crosses | Neutral |

Record:

```text
Bias = ______
```

---

# 6. Trade Plan

Write the trade plan **before entering**.

| Field  | Example |
| ------ | ------- |
| Entry  | 22460   |
| Stop   | 22410   |
| Target | 22580   |

Example trade logic:

```text
Phase = Pullback
Setup = Support Sweep
Bias = Bullish
```

Trade idea:

```text
Bull Call Spread
Buy 22450 CE
Sell 22600 CE
```

---

# 7. Execution Rules

Enter trade only when:

* Entry level triggers
* Market structure still valid

Never chase price.

---

# 8. Trade Management

Monitor conditions:

| Situation                 | Action       |
| ------------------------- | ------------ |
| Price moves +30–40 points | Lock profits |
| Momentum weakens          | Exit         |
| Target level reached      | Exit         |

---

# 9. End-of-Day Decision

At **3:10 PM** evaluate market again.

| Phase       | Decision       |
| ----------- | -------------- |
| Expansion   | Hold overnight |
| Pullback    | Partial hold   |
| Compression | Exit           |
| Transition  | Exit           |

Record decision.

---

# 10. Post-Trade Review

After market close write:

| Item         | Example                     |
| ------------ | --------------------------- |
| Trade Result | +2R                         |
| Mistake      | Late entry                  |
| Lesson       | Wait for sweep confirmation |

Continuous review improves discipline.

---

# Final Rule

Never take a trade without completing this checklist.

```text
No checklist → No trade
```
