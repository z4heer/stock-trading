# PSA System Practice Guide (Chart Replay + Paper Trade)

Goal:

```text
Train the brain to recognize structure
Practice disciplined trade planning
Build confidence in execution
```

You will **NOT place real trades** during this phase.

---

# 1. Tools Needed

You only need:

1️⃣ Chart platform (TradingView or broker chart)
2️⃣ Your **PSA Master Workbook**
3️⃣ Option chain data source (for practice)

Focus instrument:

* NIFTY 50

---

# 2. Practice Schedule

Spend **30–45 minutes daily**.

Structure:

| Step             | Activity           | Time   |
| ---------------- | ------------------ | ------ |
| Market Dashboard | Pre-market context | 5 min  |
| Chart Study      | Identify structure | 10 min |
| Trade Plan       | Fill template      | 10 min |
| Replay           | Simulate trade     | 10 min |
| Journal          | Review result      | 5 min  |

---

# 3. Step 1 — Fill Market Dashboard

Before looking deeply at chart, write context.

Example exercise values:

| Field     | Example |
| --------- | ------- |
| Spot      | 22450   |
| India VIX | 13.2    |
| PCR       | 1.05    |
| Max Pain  | 22400   |
| ATM Call  | 120     |
| ATM Put   | 110     |
| Call OI   | 22600   |
| Put OI    | 22300   |

Then calculate:

```text
Straddle = 230
Expected move = 230
```

Range:

```text
22220 – 22680
```

Now write market expectation:

```text
Market likely range / mild bullish
```

---

# 4. Step 2 — Mark Chart Levels

Open NIFTY chart and mark:

### Liquidity Levels

Draw lines at:

* Previous day high
* Previous day low
* major swing high
* major swing low

---

### Positioning Levels

Draw:

* Call OI resistance
* Put OI support

---

### Expected Move

Mark:

```text
Expected move high
Expected move low
```

Now the chart structure is clear.

---

# 5. Step 3 — Identify Market Phase

Look at price behavior.

Choose one:

| Phase       | Signs         |
| ----------- | ------------- |
| Expansion   | large candles |
| Pullback    | retracement   |
| Compression | tight range   |
| Transition  | reversal      |

Write this in the **Trade Plan sheet**.

---

# 6. Step 4 — Identify Setup

Now look for a **valid PSA trigger**.

Examples:

| Trigger         | Description    |
| --------------- | -------------- |
| Liquidity sweep | stop run       |
| VWAP reclaim    | momentum shift |
| Range sweep     | breakout trap  |

Example:

```text
Previous low sweep
VWAP reclaim
```

Setup type:

```text
Liquidity sweep breakout
```

---

# 7. Step 5 — Fill Trade Plan

Now fill the trade plan.

Example exercise:

| Field        | Value           |
| ------------ | --------------- |
| Market Phase | Expansion       |
| Bias         | Bullish         |
| Setup Type   | Liquidity Sweep |
| Entry        | 22450           |
| Stop         | 22380           |
| Target       | 22600           |

---

# 8. Step 6 — Translate to Option Trade

Find ATM option premium.

Example:

```text
22450 CE = 120
```

Stop translation:

```text
Option stop ≈ 95
```

Fill:

| Field        | Value |
| ------------ | ----- |
| Option Entry | 120   |
| Option Stop  | 95    |

---

# 9. Step 7 — Plan Spread

Define spread **before trade**.

Example:

```text
Buy 22450 CE
Sell 22600 CE
```

Write:

```text
Spread = 22450 / 22600
```

---

# 10. Step 8 — Simulate Trade

Now replay the chart forward.

Watch how price moves.

Track:

| Event                    | Action         |
| ------------------------ | -------------- |
| Price reaches 60% target | prepare spread |
| Price reaches 70% target | convert spread |
| Price hits resistance    | exit           |

Record what happens.

---

# 11. Step 9 — Journal the Result

After simulation fill the journal.

| Field        | Example |
| ------------ | ------- |
| Entry        | 22450   |
| Exit         | 22590   |
| Points       | 140     |
| Option Entry | 120     |
| Option Exit  | 200     |

Then answer:

```text
Was the setup valid?
Did the plan follow rules?
What could improve?
```

---

# 12. Weekly Review Exercise

At the end of the week review all trades.

Check:

### Pattern Success

Which setups worked best?

Example:

| Setup           | Result       |
| --------------- | ------------ |
| Liquidity sweep | high success |
| Pullback        | moderate     |

---

### Execution Errors

Common mistakes:

* early spread conversion
* wrong stop placement
* ignoring market phase
