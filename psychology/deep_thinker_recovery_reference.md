# Deep Thinker Professional Recovery Reference

## 1. Case Overview

**Context**

A professional with long corporate experience experiencing:

* reduced performance
* difficulty communicating ideas
* cognitive fatigue
* sensory sensitivity

Despite good analytical ability, the person struggles to deliver visible results.

---

# 2. Observed Symptoms

## Cognitive Symptoms

| Symptom           | Description                                   |
| ----------------- | --------------------------------------------- |
| Head heaviness    | mental fatigue or brain fog                   |
| Memory difficulty | difficulty recalling names or details         |
| Expression gap    | understands concepts but struggles to explain |

---

## Behavioral Symptoms

| Symptom            | Description                     |
| ------------------ | ------------------------------- |

| Argument avoidance | avoids conflict situations      |
| Meeting silence    | minimal speaking in discussions |

## Sensory Symptoms

| Symptom           | Description                  |

# 6. Sensory Sensitivity / Migraine Pattern

Possible indicators:

* light sensitivity
* sound sensitivity
* heavy head feeling

Possible contributing factors:

* stress
* sleep imbalance
* excessive screen exposure

Basic mitigation:

* hydration
* regular breaks
* calm environment
* controlled lighting
