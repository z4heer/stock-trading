## Template को कैसे इस्तेमाल करें

### 1️⃣ Morning

* **Mental_Log** भरें
* **Tasks sheet** में Top-3 tasks लिखें

### 2️⃣ Daytime

* Tasks का status update करें

### 3️⃣ Evening

* **Health_Log** में sleep/walking/music score डालें
* **Communication_Log** में speaking practice डालें

### 4️⃣ Weekly

* **Output_Tracker** में knowledge output add करें

### 5️⃣ Dashboard

Dashboard automatically दिखाएगा:

* energy score
* tasks completed
* communication sessions
* outputs created

## Suggested Usage Pattern

| Time    | Action                 |
| ------- | ---------------------- |
| Morning | planning               |
| Work    | task execution         |
| Evening | health + communication |
